Установка и запуск рендера

Я хочу обновить старую версию плагина на новую.

В этом случае вам нужно удалить полностью старую версию плагина, через панель управления виндовс, и установить новую.

Поставил, но не работает.

Попробуйте установить VS Redist 2017 (ссылка с сайта Microsoft).

После установки VS Redist 2017, плагины не загружаются: “error code 126 - specified module could not be found.”

Скопируйте из папки bin2, установленного рендера, три dll-ки: msvcp120.dll, msvcr120.dll, vcomp120.dll, в корень папки 3ds Max.
Например из "C:/[Hydra]/bin2" в "c:\Program Files\Autodesk\3ds Max 2019\".
Если вдруг в папке 3д макса эти dll-ки уже есть, рекомендуется их предварительно сохранить.

После нажатия Render ничего не происходит.

Обычно в этом случае, в папке “logs” установленного рендера, например “C:\[Hydra]\logs\”, в файле “plugin_log.txt”, ближе к концу можно встретить строчку “Can't connect to hydra.exe”.
Попробуйте отключить песочницу (sandbox) в вашем антивирусе. У Касперского такой механизм называется песочница. У Comodo это называется Auto sandbox. Наш рендер использует межпроцессное взаимодействие для общения между плагином и рендер-сервером hydra.exe. Песочница может блокировать этот механизм.
Попробуйте запустить 3ds max с правами Администратора.
